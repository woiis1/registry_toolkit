# registry_toolkit
Can modify registry files, support files and folders, support deletion and addition
---

## 项目说明 / Project Description
This project provides tools to modify registry files, supporting the addition, deletion, and management of files and folders.  
本项目提供注册表文件修改工具，支持文件与文件夹的添加、删除和管理操作。
